# Paso 2 - Formulario y navegación

## Español

### Objetivo
Agregar el formulario de login y navegación a `chat.html`.

### Archivos a copiar
- `login.html`
- `chat.html`

Copiar desde `steps/step-2` hacia `c02-html-start`.

### Validación
1. Abrir `login.html` en navegador.
2. Verificar campos `Username` y `Password`.
3. Presionar `Login` y confirmar que abre `chat.html`.

## English

### Objective
Add the login form and navigation to `chat.html`.

### Files to copy
- `login.html`
- `chat.html`

Copy from `steps/step-2` into `c02-html-start`.

### Validation
1. Open `login.html` in the browser.
2. Verify `Username` and `Password` fields.
3. Press `Login` and confirm it opens `chat.html`.
