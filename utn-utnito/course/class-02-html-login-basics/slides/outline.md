## Slide 2 - Objetivo de la clase
- Construir un login simple con HTML y CSS.
- Entender estructura semántica mínima.
- Preparar base visual para el chat de la Clase 03.

## Slide 3 - Qué conceptos vamos a ver hoy
- Estructura base HTML: define el esqueleto de cualquier página.
- Etiquetas semánticas (`main`, `section`, `header`, `footer`): ordenan el contenido y mejoran legibilidad.
- Formularios (`form`, `label`, `input`, `button`): capturan datos del usuario.
- CSS con clases (`.login-card`) e id (`#username`): cuándo usar cada uno.
