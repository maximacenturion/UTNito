# Referencia rápida - Validación HTML y Navegación

## Español

### Validaciones HTML nativas
Ejemplos útiles:

```html
<input type="text" required minlength="3" maxlength="20" />
<input type="email" required />
<input type="number" min="16" max="120" />
<input type="password" minlength="6" maxlength="20" />
<input type="text" pattern="[A-Za-z0-9._-]+" />
```

### Link vs Button

```html
<a href="chat.html">Enter chat</a>
<button type="submit">Login</button>
<button type="button">Composer action demo</button>
```

- `a`: navegación.
- `button`: acción (enviar form o disparar comportamiento).
