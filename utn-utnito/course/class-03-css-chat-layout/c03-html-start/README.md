# c03-html-start

## Español

Proyecto inicial para la Clase 03.
Incluye:
- `login.html` (base para repaso de forms + validación + navegación),
- `chat.html` (base mínima para empezar maquetación),
- `chat.css` (archivo inicial),
- `login.css`, `tokens.css` y assets.

### Uso
1. Abrir esta carpeta en VS Code.
2. Abrir `login.html` y `chat.html` en navegador con doble click.
3. Aplicar cambios copiando archivos de `../steps/` en orden.

## English

Initial project for Class 03.
Includes:
- `login.html` (base for forms/validation/navigation review),
- `chat.html` (minimum base to start layout),
- `chat.css` (initial file),
- `login.css`, `tokens.css`, and assets.

### Usage
1. Open this folder in VS Code.
2. Open `login.html` and `chat.html` in browser by double click.
3. Apply changes by copying files from `../steps/` in order.
