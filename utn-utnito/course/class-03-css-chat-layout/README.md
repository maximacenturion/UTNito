# Clase 03 - CSS + Chat Layout (HTML/CSS)

## Español

### Objetivo
Construir la pantalla principal del chat de UTNito con HTML y CSS, de forma incremental y conceptual.

### Duración sugerida
60 minutos

### Enfoque de clase
En esta clase no se usa JavaScript.
El foco está en:
- estructura HTML de una interfaz real,
- validaciones básicas de formularios del navegador,
- navegación entre páginas,
- fundamentos de CSS,
- box model,
- Flexbox básico.

### Estructura de la clase
- `c03-html-start`: proyecto inicial.
- `steps/step-1`: login review + validación + navegación + link vs button.
- `steps/step-2`: estructura base de `chat.html`.
- `steps/step-3`: introducción CSS + selectores básicos.
- `steps/step-4`: demo aislada de box model.
- `steps/step-5`: demo aislada de Flexbox.
- `steps/step-6`: aplicar Flexbox al chat + cierre visual.
- `c03-html-end`: resultado final esperado.
- `slides/outline.md`: guía para armar presentación.
- `checklist.md`: validación de cierre.
- `troubleshooting.md`: errores comunes.

### Flujo de práctica
1. Clonar repositorio:
   - `git clone https://github.com/matmatys/programacion_III_UTN.git`
2. Entrar a la carpeta inicial:
   - `cd programacion_III_UTN/utn-utnito/course/class-03-css-chat-layout/c03-html-start`
3. Abrir `c03-html-start` en VS Code.
4. Seguir los `steps` en orden y copiar los archivos indicados.
5. Validar cada paso en navegador con doble click (sin servidor ni framework).

### Entregable de clase
- `chat.html`
- `chat.css`

Con estas dos piezas el alumno ya tiene el layout del chat sin lógica dinámica.

## English

### Objective
Build the main UTNito chat screen with HTML and CSS using an incremental and conceptual approach.

### Suggested Duration
60 minutes

### Class Approach
No JavaScript is used in this class.
Focus:
- HTML structure for a real UI,
- browser native form validation basics,
- page navigation,
- CSS fundamentals,
- box model,
- basic Flexbox.

### Class Structure
- `c03-html-start`: initial project.
- `steps/step-1`: login review + validation + navigation + link vs button.
- `steps/step-2`: base `chat.html` structure.
- `steps/step-3`: CSS introduction + basic selectors.
- `steps/step-4`: isolated box model demo.
- `steps/step-5`: isolated Flexbox demo.
- `steps/step-6`: apply Flexbox to chat + visual closure.
- `c03-html-end`: expected final result.
- `slides/outline.md`: slide preparation guide.
- `checklist.md`: closing validation.
- `troubleshooting.md`: common issues.

### Practice Flow
1. Clone repository:
   - `git clone https://github.com/matmatys/programacion_III_UTN.git`
2. Go to initial folder:
   - `cd programacion_III_UTN/utn-utnito/course/class-03-css-chat-layout/c03-html-start`
3. Open `c03-html-start` in VS Code.
4. Follow `steps` in order and copy required files.
5. Validate each step in browser by double click (no server/framework).

### Class Deliverable
- `chat.html`
- `chat.css`

With these two files students complete the static chat layout without dynamic logic.
