# Step 1 - Login review + validation + navigation + link vs button

## Español

### Objetivo
Hacer un repaso rápido del login y explicar en vivo validaciones HTML nativas del navegador, navegación entre páginas y diferencia conceptual entre link y button.

### Archivos a copiar
- `login.html`

Copiar desde `steps/step-1` hacia `c03-html-start`.

### Conceptos a explicar
- `required`
- `type="email"`
- `type="number"`
- `minlength` / `maxlength`
- `pattern`
- `<a href="...">` para navegación
- `<button>` para acción

### Experimentos sugeridos
- Enviar el form con campos vacíos.
- Ingresar email inválido.
- Escribir texto en campo numérico.
- Comparar `Enter chat with link` vs `Login (Button)`.
